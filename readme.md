### CNN 인공 신경망

입력층(input layer) -> 은닉층(hidden layer) -> 출력층(output layer)

은닉층이 2층 이상일 때는 심층 신경망(DNN)

### RNN 인공 신경망

반복 신경망. 이는 문장 인식에 뛰어남. 이 신경망 또한 입력 -> 은닉 -> 출력층으로 구성.

## 딥러닝

### 핵심은 순전파, 역전파.<br><br> 순전파는 입력,편향이 예측으로 흘러간다면, 역전파는 예측이 입력, 편향으로 역으로 흐르는 개념

### 순전파

<img width="532" height="276" alt="image" src="https://github.com/user-attachments/assets/f304e804-d1cc-44dd-91fd-a51018c5112d" />

x : 입력값, y : 예측값, b : 편향값, w : 가중치

### 평균 제곱 오차

E = (y-yT)^2 / 2

E : 오차, y : 예측값, yT : 목표값 또는 라벨.

여기서 오차는 손실 또는 비용. 오차값이 작을 수록 좋은 인공 신경망.

### 역전파 오차

yE = y - yT

yE : 역전파 오차, y : 예측값, yT : 목표값 또는 라벨

yE는 y에 대한 오차 E의 순간 변화율을 의미, 편미분을 통해 유도됨. // 평균 제곱 오차를 y에 대해 편미분하면 나옴.

### 입력 역전파

xE = yE*w

xE : 입력 역전파, yE : 역전파 오차, w : 가중치

### 가중치, 편향 순정파

순전파에서 x와 w, b와 1의 위치를 뒤바꿔줌. 교환법칙이 성립한다면 순전파의 공식과 같음.

### 가중치, 편향 역전파

y -> yE, w -> wE, b -> bE로 변경. 그리고 x, 1 --> y가 아닌, y -> x, 1로 가는걸로 변경.

그러면, wE = yE*x, bE = yE*1이 됨.

### 신경망 학습

w = w - lr*wE

b = b - lr*bE

lr : 학습률

최종적으로, 편향값과 가중치를 매 학습마다 갱신하면서 예측값이 목표값에 도달하도록 값이 조절됨.

또한 이 방법을 **경사하강법**이라고 함.

## 작성법

예를 들어서, 예측값 2개, 입력값이 2개 있다고 하자.

x1, x2, 1을 만들고, 각각에 대해 w1, w2, b1, w3, w4, b2가 있다.

**y1 = x1w1 + x2w2 + 1b1**, **y2 = x1w3 + x2w4 + 1b2**

**E = ((y1 -y1T)^2 + (y2 - y2T)^2)) / 2**

**y1E = y1 - y1T**, **y2E = y2 - y2T**

**w1E = y1E*x1 w2E = y1E*x2 b1E = y1E*1**, **w3E = y2E*x1 w4E = y2E*x2 b2E = y2E*1**

**w1 = w1 - Ir*w1E w2 = w2 - Ir*w2E b1 = b1 - Ir*b1E**, **w3 = w3 - Ir*w3E w4 = w4 - Ir*w4E b2 = b2 - Ir*b2E** 

이렇게 작성하면 된다.
